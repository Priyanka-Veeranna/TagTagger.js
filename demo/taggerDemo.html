TagTagger.js Demo

Demonstrates TagTagger.js with default settings

➨ Default look and feel comes with:

  • ✓ wrapper box fixed with 500px width and min-height of 90px
  • ✓ Tag element with dark blue background and white text color
  • ✓ Minimum length of tag text is 1
  • ✓ Duplicate tag message Showing yellow background
  • ✓ Styling changeable by modifying taggerStyles.css
